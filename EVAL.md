# clarify 스킬 평가 가이드

## 평가 구조: 3 Layer

### Layer 1: 메커니즘 준수 (자동 판정 가능)

| # | 규칙 | 확인 방법 | 근거 |
|---|------|----------|------|
| 1 | Phase 0에서 반영(mirror) 실행 | 첫 AI 응답에 사용자 말 되비추기 존재 | FBI BCSM |
| 2 | 진입 Phase 정확 | 사용자 신호 → 올바른 Phase 라우팅 | 상태 감지 |
| 3 | 첫 질문이 열린 질문 (Phase 1) | Phase 1 턴1에 선택지 번호 없음 | PEACE 모래시계 |
| 4 | 한 턴에 질문 하나 | 응답당 "?" 2개 이하 | 인지부하 |
| 5 | 전제도전 전 반영+요약 존재 | 전제도전 직전에 요약 문장 존재 | FBI BCSM 순서 |
| 6 | 전제도전 근거가 이전 답변에서만 유래 | 재프레이밍 키워드 ⊆ Q1-Q2 답변 키워드 | MI 교정반사 억제 |
| 7 | 코드 금지 | 코드 블록(```) 없음 | 사고 도구 원칙 |
| 8 | "안 하기" 선택지 포함 | Phase 2 대안에 포기 옵션 존재 | MI 자율성 |
| 9 | 5턴 이내 종료 | Phase 2 질문이 5개 초과하지 않음 | 피로 방지 |
| 10 | 모호성 대시보드 생성 (Phase 3) | 4영역 점수 테이블 존재 | spec-clear 계승 |

**합격 기준:** 10개 중 8개 이상 준수

### Layer 2: 사고 전환 품질 (사람 판단)

**2-1. 입력-출력 전환 거리**

| 점수 | 기준 | 예시 |
|------|------|------|
| 0 | 변화 없음 | "리포트 자동화" → "리포트 자동화" |
| 1 | 구체화 | "리포트 자동화" → "Athena 쿼리 자동화 + 슬랙 전송" |
| 2 | 재프레이밍 | "리포트 자동화" → "캠페인 이상 감지 알림" |
| 3 | 전환 | "리포트 자동화" → "운영팀 의사결정 지원 체계" |

목표: 평균 1.5 이상. 단, 0도 성공일 수 있음 (원래 방향이 맞았을 때).

**2-2. 사용자 자기 평가 3문항**

스킬 종료 후:
1. "이 대화 전후로 확신이 얼마나 달라졌나요?" (1-5)
2. "미처 생각 못 했던 것을 발견했나요?" (예/아니오 + 내용)
3. "다음에 할 행동이 명확한가요?" (1-5)

목표: Q1 ≥ 3.5, Q2 예율 ≥ 60%, Q3 ≥ 4.0

**2-3. 과제 실행률 (후행 지표)**

1주 후 확인: 실행(1.0) / 수정 실행(0.5) / 미실행(0.0)
목표: ≥ 0.7

### Layer 3: 학술 기반 품질 (신규)

| 지표 | 측정 | 근거 |
|------|------|------|
| 반영 정확도 | 사용자가 반영에 "맞아요" 반응 비율 | FBI BCSM 라포 |
| 질문 유형 적응성 | 모호도와 질문 유형(열린/닫힌)의 일치도 | PEACE 모래시계 |
| 교정반사 억제율 | AI가 먼저 해결책을 제시한 횟수 = 0 | MI OARS |
| 암묵지 추출 여부 | "초보자가 놓치는 것" 유형 발견 존재 | ACTA |

## 기존 스킬 대비 A/B 비교 방법

```
같은 시나리오를:
A) clarify 스킬 활성화 상태에서 실행
B) 기존 think-clear/product-think/spec-clear 중 해당 스킬로 실행
C) 스킬 없이 빈 Claude에서 실행

비교 항목:
- 반영(mirror) 발생 여부 (A만? B/C도?)
- 첫 질문 유형 (열린 vs 닫힌)
- 전제도전 타이밍 (라포 후 vs 즉시)
- 과제의 구체성과 크기
- 전체 턴 수 (효율성)
- 사용자 자기 평가 점수 차이
```

## 핵심 원칙

1. **Layer 1이 통과해야 Layer 2가 의미 있다** — 규칙 없는 좋은 결과는 우연
2. **5회 수동 평가 후 패턴을 찾아 스킬 수정** — 자동화는 그 다음
3. **전환 거리 0도 성공** — 무리한 재프레이밍은 해로움
4. **과제 실행률이 궁극적 지표** — 대화가 좋아도 행동이 없으면 실패
5. **Layer 3은 v2 이후** — 먼저 Layer 1-2로 안정화한 후 학술 지표 추가
